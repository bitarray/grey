# Grey

Grey has moved to the [JAR monorepo](https://github.com/jarchain/jar/tree/master/grey).

## What is Grey?

Grey is a JAM (Join-Accumulate Machine) blockchain node implementation in Rust, following the [JAR specification](https://github.com/jarchain/jar/tree/master/spec). It features 400+ tests, 101/101 conformance blocks, a multi-node testnet with GRANDPA finality, and an x86-64 PVM recompiler that outperforms polkavm on all workloads.
